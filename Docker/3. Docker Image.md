# 撰寫 Dockerfile
### Download julia
```sh
[lvmh@localhost]$ mkdir /home/lvmh/Desktop/temp
[lvmh@localhost]$ cd /home/lvmh/Desktop/temp
[lvmh@localhost]$ wget https://julialang-s3.julialang.org/bin/linux/x64/1.0/julia-1.0.0-linux-x86_64.tar.gz
# 下載時間久
```

### 撰寫 Dockerfile
```sh
"""dockerfile permission denied??"""
[lvmh@localhost]$ vi Dockerfile
FROM ubuntu:18.04
RUN apt update && apt upgrade
RUN tar xvfz julia-1.0.0-linux-x86_64.tar.gz -C /opt
RUN ln -s /opt/julia-1.0.0/bin/julia /usr/bin/julia

[lvmh@localhost]$ docker build --no-cache -t julia .
```
